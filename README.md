# Mapa Mental Educativo

Ferramenta web gratuita para **organizar pesquisas escolares** por meio de mapas mentais interativos. Desenvolvida para apoiar o **Ensino Fundamental I** (1º ao 5º ano), com foco em competências da **BNCC** e da **BNCC de Computação (BNCC Tecnologia)**.

**Acesse online:** [https://mapa-mental.educar.workers.dev](https://mapa-mental.educar.workers.dev)

---

## O que é esta ferramenta?

O **Mapa Mental Educativo** permite que crianças e professoras organizem ideias de forma visual durante uma pesquisa escolar. O tema central fica no meio do mapa; ramificações representam perguntas, categorias, descobertas e fontes consultadas.

A ferramenta funciona no navegador (computador, tablet ou celular), **não exige cadastro** e oferece:

- Criação e edição de tópicos com duplo clique
- Cores automáticas para diferenciar ramos
- Arrastar tópicos e navegar com zoom
- **Salvar na nuvem** e gerar link para compartilhar
- **Baixar** e **abrir** arquivos JSON (backup local)
- Auto-salvamento após alterações

| Perfil | Como usar |
|--------|-----------|
| **Alunos (6 a 10 anos — EF I)** | Organizar pesquisas de ciências, história, geografia e projetos interdisciplinares |

### Por que o Ensino Fundamental I?

Nessa faixa etária, as crianças estão desenvolvendo:

- **Leitura e escrita** funcional (registrar ideias com palavras-chave)
- **Pensamento lógico** (relacionar causa, consequência e categorias)
- **Autonomia na pesquisa** (formular perguntas e buscar respostas)
- **Primeiras competências digitais** (usar ferramenta web de forma guiada)

O mapa mental traduz essas habilidades em uma interface visual, concreta e lúdica — adequada à cognição infantil.

## BNCC — Base Nacional Comum Curricular

A **BNCC** define o que todos os estudantes brasileiros devem aprender ao longo da Educação Básica. Para o **Ensino Fundamental – Anos Iniciais**, a ferramenta dialoga especialmente com:

### Linguagens

- **(EF15LP01)** Reconhecer que textos atendem a diferentes propósitos
- **(EF15LP03)** Localizar informações explícitas em textos
- **(EF15LP08)** Organizar informações selecionadas para compor texto

**Indicação:** usar ramos do mapa como rascunho antes de produzir um texto expositivo ou relatório de pesquisa.

### Matemática — Raciocínio lógico

- **(EF01MA01)** Utilizar números naturais como indicadores de quantidade ou ordem
- **(EF03MA01)** Ler, escrever e comparar números naturais
- **(EF04MA01)** Ler, escrever e ordenar números naturais

**Indicação:** incluir ramos com dados numéricos (população, datas, medidas) e comparar quantidades entre categorias.

### Ciências da Natureza

- **(EF02CI01)** Identificar de que materiais são feitos objetos do cotidiano
- **(EF03CI01)** Produzir diferentes sons a partir de objetos
- **(EF04CI01)** Identificar misturas na vida diária
- **(EF05CI01)** Construir proposições sobre regularidades em fenômenos naturais

**Indicação:** pesquisas sobre animais, plantas, corpo humano, energia e meio ambiente com ramos como *O que é?*, *Onde vive?*, *Por que é importante?*

### Ciências Humanas

- **(EF02GE01)** Identificar e comparar aspectos de diferentes lugares
- **(EF03HI01)** Identificar os grupos populacionais que formam a cidade
- **(EF04HI01)** Reconhecer a história como resultado da ação de diferentes grupos

**Indicação:** mapas sobre comunidade, bairro, tradições locais, personagens históricos e linha do tempo simplificada.

### Computação na BNCC (Computação na Educação Básica)

Desde 2022, a BNCC incorpora **Computação** como componente transversal. No EF I, destacam-se:

- **Pensamento computacional:** decompor problemas, reconhecer padrões, representar informações
- **Cultura digital:** usar tecnologias de forma ética, segura e produtiva
- **Mundo digital:** compreender que dados podem ser organizados e compartilhados

## BNCC Tecnologia (Computação)

A expressão **BNCC Tecnologia** refere-se ao conjunto de **competências e habilidades de Computação** previstas na BNCC, muitas vezes trabalhadas em conjunto com a **BNCC Computação** (documento específico do MEC/CEB).

### Competências relacionadas

| Competência | Relação com o Mapa Mental |
|-------------|---------------------------|
| **Pensamento computacional** | Decompor um tema em sub-tópicos; identificar relações hierárquicas |
| **Autoria e criatividade digital** | Criar e editar um artefato digital original (o mapa) |
| **Comunicação e colaboração** | Compartilhar link do mapa com colegas e professora |
| **Cidadania digital** | Discutir o que pode ou não ser copiado; citar fontes nos ramos |
| **Dados e informação** | Organizar, nomear e estruturar informações de forma significativa |

### Habilidades computacionais (EF I) apoiadas pela ferramenta

- Representar informações de forma estruturada (tópico central → ramificações)
- Sequenciar passos de uma investigação (perguntas → busca → registro)
- Manipular uma interface digital (clicar, arrastar, editar, salvar)
- Persistir e recuperar dados (salvar na nuvem, abrir link, baixar JSON)

## Indicações de uso pedagógico

### 1. Pesquisa guiada (2º ou 3º ano)

**Tema:** Animais do bioma local

1. Professora apresenta o tema central: *"Animais da nossa região"*
2. Alunos, em duplas, criam ramos: *Mamíferos*, *Aves*, *Onde vivem?*, *O que comem?*
3. Cada dupla salva o mapa e compartilha o link
4. Fechamento: comparar mapas e complementar informações

**Competências:** investigação científica, organização de informação, colaboração digital.

### 2. Projeto interdisciplinar (4º ou 5º ano)

**Tema:** Água — recurso essencial

Ramos sugeridos:

- Ciências: ciclo da água, contaminação
- Geografia: rios e bacias da região
- História: uso da água ao longo do tempo
- Matemática: consumo médio por dia
- Ação: o que podemos fazer para economizar?

**Produto final:** texto multimodal ou apresentação oral baseada no mapa.

### 3. Pré-leitura e pré-escrita

Antes de escrever um texto, o aluno monta o mapa com palavras-chave. Isso reduz bloqueio de escrita e explicita a estrutura do texto (introdução → desenvolvimento → conclusão).

### 4. Avaliação formativa

A professora pode observar:

- Capacidade de formular perguntas relevantes
- Organização lógica das ideias
- Uso adequado da ferramenta digital
- Evolução do mapa ao longo de etapas da pesquisa

## Como usar (passo a passo)

1. **Abra** a ferramenta no navegador
2. **Duplo clique** no tópico central para definir o tema da pesquisa
3. Clique no botão **+** de um tópico para adicionar sub-ideias
4. **Duplo clique** em qualquer tópico para editar o texto
5. **Arraste** tópicos para reorganizar; use scroll ou botões **+ / −** para zoom
6. Clique em **Salvar** para guardar na nuvem e obter um link compartilhável
7. Use **Baixar** para guardar uma cópia no computador; **Abrir** para recuperar um arquivo salvo
8. Clique em **Ajuda** dentro da aplicação para rever instruções rápidas

### Dicas para crianças

- Use frases curtas ou palavras-chave
- Cores diferentes ajudam a separar assuntos
- Comece pelo que você já sabe; vá completando com a pesquisa
- Peça ajuda da professora para salvar e copiar o link

## Desenvolvimento local

### Pré-requisitos

- [Node.js](https://nodejs.org/) 18+
- Conta [Cloudflare](https://dash.cloudflare.com/) (para deploy)
- [Wrangler CLI](https://developers.cloudflare.com/workers/wrangler/) v4+

### Instalação

```bash
git clone https://github.com/dduenhas/mapa-mental.git
cd mapa-mental
npm install
```

### Criar namespace KV (primeira vez)

```bash
npx wrangler kv namespace create MINDMAPS
```

Copie o `id` gerado para `wrangler.toml` no binding `MINDMAPS`.

### Executar localmente

```bash
npm run dev
```

Acesse `http://localhost:8787`.

### Deploy na Cloudflare

```bash
npm run deploy
```

## Arquitetura técnica

```
mapa-mental/
├── public/           # Frontend estático (HTML, CSS, JS)
│   ├── index.html
│   ├── app.js        # Lógica do mapa mental (ES modules)
│   └── style.css
├── src/
│   └── index.js      # Cloudflare Worker (API + assets)
├── wrangler.toml     # Configuração do Worker e KV
└── package.json
```

| Camada | Tecnologia | Função |
|--------|------------|--------|
| Frontend | HTML, CSS, JavaScript vanilla | Interface interativa do mapa |
| Backend | Cloudflare Workers | API REST para mapas |
| Armazenamento | Cloudflare KV | Persistência dos mapas na nuvem |
| Hospedagem | Cloudflare Workers + Assets | Deploy global, baixa latência |

### API

| Método | Rota | Descrição |
|--------|------|-----------|
| `GET` | `/api/maps/:id` | Carrega um mapa salvo |
| `PUT` | `/api/maps/:id` | Salva ou atualiza um mapa |
| `DELETE` | `/api/maps/:id` | Remove um mapa |

Validações no servidor: até 100 tópicos, textos limitados, exatamente um tópico raiz.

## Privacidade e uso responsável

- Não há cadastro de usuários; mapas são identificados por ID aleatório na URL
- Quem possui o link pode visualizar e editar o mapa — **compartilhe apenas com pessoas de confiança**
- Recomenda-se orientar alunos a **não incluir dados pessoais** (nome completo, endereço, telefone) nos tópicos
- Professores podem usar mapas anônimos ou com apelidos para atividades em sala
