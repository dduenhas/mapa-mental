Como usar o Mapa Mental

Passo a passo

  1. Clique no tópico central e dê um duplo clique para escrever o tema da pesquisa.
  2. Use o botão + em cada tópico para adicionar ideias relacionadas.
  3. Arraste os tópicos para organizar o mapa da forma que fizer mais sentido.
  4. Clique em Salvar para guardar na nuvem e gerar um link para compartilhar.

Dicas para pesquisa

  • Coloque perguntas nos ramos: O que é?, Onde vive?, Por quê?
  • Use cores diferentes para separar categorias (lugares, datas, curiosidades).
  • Comece pelo que você já sabe e vá expandindo com novas descobertas.

Atalhos

  • Duplo clique — editar tópico
  • Scroll — zoom
  • Arrastar fundo — mover o mapa
